# Flask DevOps Project 🚀

A containerized Flask web application built with Docker, automated using GitHub Actions CI/CD, and deployed on Railway cloud platform.

---

## 📌 Project Overview

This project demonstrates a complete beginner-friendly DevOps workflow including:

- Flask web application development
- Docker containerization
- CI/CD automation using GitHub Actions
- Cloud deployment using Railway
- Linux-based development environment (WSL Ubuntu)

---

## 🛠️ Tech Stack

- Python
- Flask
- Docker
- GitHub Actions
- Railway
- Linux (WSL Ubuntu)
- Git & GitHub

---

## 🚀 Features

- Lightweight Flask application
- Dockerized deployment workflow
- Automated CI pipeline with GitHub Actions
- Cloud deployment with Railway
- Version-controlled project using Git
