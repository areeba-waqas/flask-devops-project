📌 About This Project

This Flask DevOps project demonstrates a complete modern deployment workflow using Docker containerization, GitHub Actions CI/CD automation, Railway cloud deployment, and Linux-based development environment.

🛠️ Technologies Used

Flask Docker GitHub Actions Railway Linux Python Git CI/CD

🐳 Docker Integration

The application is fully containerized using Docker to ensure consistent deployment across different environments. 

docker build -t flask-devops-project .
docker run -d -p 5000:5000 flask-devops-project

⚙️ CI/CD Pipeline

GitHub Actions automatically builds the Docker image whenever new code is pushed to the main branch.

Code Push
GitHub Actions
Docker Build
Railway Deploy
